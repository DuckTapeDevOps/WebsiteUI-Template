# Template Setup Guide

> **Note:** This guide will be completed in later phases. Currently, this template still contains NavalNomad-specific content.

## Overview

This template provides a React + Vite + TypeScript website with AWS infrastructure (S3 + CloudFront + Route53) deployment.

## Using This Template

1. Click "Use this template" on GitHub
2. Create your new repository
3. Clone and customize (see sections below)

## What to Customize

### Brand Configuration
- Brand name and tagline
- Colors and fonts
- Logo and hero images
- Navigation items

### Content
- Page text and descriptions
- Feature lists
- Contact information

### Infrastructure
- Domain name
- S3 bucket names
- GitHub repository references
- Resource names
